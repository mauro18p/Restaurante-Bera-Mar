# Restaurante-Bera-Mar
Este site apresenta um restaurante e é constituido por 4 páginas HTML 5 e utiliza Bootstrap 5.3v para estilização.

As 4 páginas são:
- *Home page ou página inicial*
- *Menu/cardápio*
- *Contacto*
- *Sobre Nós*

Este site não apresenta nenhuma depêndencia por parte do usuário, podendo ser usado apartir de qualquer browser que suporte Bootstrap.
